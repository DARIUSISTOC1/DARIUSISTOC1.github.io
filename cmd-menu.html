CMD-MENUv2

Un menu Windows en batch inspiré de ChrisTitus Tech's WinUtil. Version 2.1.6.5 beta, développé sur environ 3 ans, avec des outils système, mini-jeux, tweaks Windows et un installeur de logiciels.

Batch Windows PowerShell Python Automatisation

C'est quoi

Un menu batch interactif pour Windows avec des dizaines de fonctionnalités regroupées : outils système, nettoyage, réparation, réseau, tweaks avec états toggle, et un installeur de 30+ logiciels via winget.

Fonctionnalités

  • Installeur multi-page avec 30+ programmes (winget)
  • Mini-jeux : morpion, calcul mental, devinettes
  • Tweaks Windows avec états on/off persistants
  • Outils réseau et réparation système
  • Création de points de restauration
  • Affichage RAM via PowerShell intégré
  • Génération du code via scripts Python

Ce que j'ai appris

3 ans de développement itératif. Ce projet m'a appris à gérer la complexité d'un gros script, à débugguer des problèmes d'encodage (CRLF, BOM), à automatiser la génération de code et à concevoir une UX dans un terminal.
