C'est quoi

Un menu batch interactif pour Windows avec des dizaines de fonctionnalités : outils système, nettoyage, réparation, réseau, tweaks avec états toggle, installeur de 30+ logiciels via winget. Version 2.1.6.5 beta, développé sur ~3 ans.

Fonctionnalités

  • Installeur multi-page 30+ programmes (winget)
  • Mini-jeux : morpion, calcul mental, devinettes
  • Tweaks Windows avec états on/off persistants
  • Outils réseau et réparation système
  • Création de points de restauration
  • Affichage RAM via PowerShell
  • Génération du code via scripts Python

Ce que j'ai appris

3 ans de développement itératif. Encodage (CRLF, BOM), génération de code automatisée, UX dans un terminal.
